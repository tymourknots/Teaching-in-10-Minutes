Hard Coding Example:

Task: Convert the list comprehension that filters out all odd numbers from a list and squares them into a for-loop with conditionals.

list_comprehension = [x**2 for x in range(10) if x % 2 == 0]

Answer:

squared_evens = []

for number in range(10):

   if number % 2 == 0:

      squared_evens.append(number**2)

What are list comprehensions in Python?

A list comprehension is a concise way to create lists in Python. It offers a shorter syntax when you want to create a new list based on the values of an existing list or any iterable. Instead of using loops and appending to a list, you can achieve the same using a single line of code with list comprehension.

Basic syntax of a list comprehension:

[new_element for item in iterable if condition]

Example without:

squares = []
                for x in range(1, 11):
                    if x % 2 == 0:
                        squares.append(x**2)
                        print(squares)

Example with:

squares = [x**2 for x in range(1, 11) if x % 2 == 0]

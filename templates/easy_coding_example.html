Task: Create a list of the cube of each even number from 2 to 10 using list comprehension.

cubes = [x**3 for x in range(2, 11) if x % 2 == 0]

This will produce a list of the cubes of all even numbers between 2 and 10.

Output: [8, 64, 216, 512, 1000]
